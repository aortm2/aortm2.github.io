Q
연구출판물이 안열려요. 어떻게 해야 하나요?
A 교육원 연구출판물은 브라우져의 환경을 맞춰주셔야 합니다.
브라우저 환경을 다음과 같이 설정해 주십시요.
1. 브라우저의 도구 - 인터넷옵션 클릭
2. 보안탭 클릭
3. 신뢰할수 있는 사이트 클릭 후 하단의 사이트 버튼 클릭
4. https:// 확인필요 옵션에 체크를 해제한후 다음 세개의 주소를 추가 후 확인버튼 클릭
*.keli.kr
http://www.keli.kr
5. 이 영역에 적용할 보안수준에서 슬라이드 바를 최소로 내려준후 적용버튼과 확인버튼 클릭(슬라이드 바가 나오지 않을경우 기본수준 버튼을 클릭하면 슬라이드 바가 나옵니다.)
6. 페이지를 새로고침 실행 후 간행물에서 연구출판물 조회
